
# Workshop-KI-Prototyp

## Beschreibung
Eine einfache HTML/JS-Webanwendung, die mit Hilfe der OpenAI API automatisch Workshoppläne erstellt.

## Nutzung
1. Öffne `index.html` im Browser.
2. Trage deinen OpenAI API-Key in `script.js` ein (Zeile mit `DEIN_API_KEY_HIER_EINFÜGEN`).
3. Fülle das Formular aus und erhalte automatisch generierte Workshopvorschläge.
